# 📤 块快速流转 (Block Flow)

一款让你能够通过右键块菜单，极速且优雅地将内容进行跨页面流转整理的效率工具。

## ✨ 功能特性

- **多目标流转**:
  - 📅 **今日日志 (Today's Journal)** — 当天的 Daily Note。
  - 🌅 **明日日志 (Tomorrow's Journal)** — 明天的 Daily Note（非常适合进行任务延期、明天待办规划）。
  - 📥 **自定义收件箱 (Custom Inbox)** — 通过标签（默认 `#收件箱`）检索或自动创建的通用收件箱页面。
- **双维度整理模式**:
  - ✂️ **物理移动 (Move)** — 将选中块物理剪切移动到目标容器末尾。
  - 🔗 **发送引用 (Send Ref)** — 保持原块不动，仅在目标容器下生成指向该原块的块引用（`{ t: "r", v: blockId }`），便于归纳快捷方式。
- **多选支持**: 支持通过框选或多选后一次性打包流转多个块。
- **个性化设置**: 可在 Sub-plugin Settings 中针对六大流转子命令分别进行开启/关闭，或者更改你的自定义收件箱标签。

## 🚀 使用方法

1. 在 Sub-plugin Settings 中启用 **Block Flow** 插件。
2. 在任意文档中，选中一个或多个块，鼠标右键唤出右键菜单。
3. 在菜单下方找到 `快速流转到...` 的各子选项（如 `移动至今日日志`、`发送引用至收件箱`）。
4. 点击后，块将被瞬间转移或引用至目标，且在屏幕右上方弹出美丽的成功通知。
