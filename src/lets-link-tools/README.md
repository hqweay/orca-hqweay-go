# 🔗 链接工具 (Link Tools)

右键块链接或块引用时，在虎鲸笔记原生菜单中注入快捷转换选项。

## ✨ 功能特性

- **右键块链接**：注入以下转换选项：
  - 🔗 **转换为块引用** — 将超链接无损转换为原生块引用，保留文本作为别名。

- **右键块引用**：注入以下转换选项：
  - 🔗 **转换为块链接** — 将块引用批量转换为常规超链接。
  - 📌 **转换为文本📌** — 在别名末尾添加 `📌` 后缀，形成更轻量的文本图钉引用。
  - 📌 **转换为📌** — 将别名设为单个 `📌`，形成纯图标图钉引用。

- **智能识别**：仅在右键块链接（`orca-note://xxx/block?blockId=xxx`）或块引用（`data-type="r"`）时才注入菜单项，普通链接不受影响。

## 🚀 使用方法

1. 在 Sub-plugin Settings 中启用 **Link Tools** 插件。
2. 在编辑器中找到一个块链接或块引用。
3. 右键点击，菜单中将出现对应的转换选项。
4. 点击即可完成转换，菜单自动关闭。

## ⚠️ 注意事项

- 本插件通过 DOM 注入实现，依赖虎鲸笔记的菜单结构，后续版本更新可能导致失效。
- 转换逻辑复用自 `lets-block-tools` 插件。
