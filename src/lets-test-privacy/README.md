# lets-privacy (隐私块)

一个极简但极其有效的隐私保护插件。它的核心理念不是“加密文件”，而是利用虎鲸笔记的底层机制，向内置 AI 隐藏敏感内容。

## 功能特性 (Features)
- **拦截 AI 读取**: AI 请求当前笔记上下文、搜索或总结时，只会读取到 `[🔒 隐私内容已隐藏]`。
- **保留原生体验**: 隐私块保留了原生的富文本编辑器功能。你依然可以加粗、高亮，甚至是嵌入子块，丝毫不影响日常书写体验。
- **无密码干扰**: 没有繁琐的密码弹窗和加密解密等待时间。

## 使用方法 (Usage)
1. 选中任意块（或将光标放置在块内）。
2. 触发块级菜单（或右键）。
3. 选择 **"设为/取消隐私块"**。
4. 块左侧将出现红色的指示条，并且右侧出现 🔒 锁形图标，代表该块的上下文已被混淆，AI 将无法读取到该块内的明文。

## 实现原理 (Under the hood)
这个插件并未真正对数据库中的内容进行加密。它使用虎鲸笔记的 `orca.renderers.registerBlock` 保留了块的 `Editable Flag` 为 `true`，以实现原生的输入体验。同时，通过 `orca.converters.registerBlock` 拦截了 "plain text" 和 "markdown" 的转换过程。当应用内 AI（或者导出功能）请求块上下文时，转换器会输出预设的伪造字符，从而切断 AI 获取你敏感数据的路径。
