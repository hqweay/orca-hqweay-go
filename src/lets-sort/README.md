# lets-sort 插件文档

本插件提供多选块排序功能，旨在帮助用户快速整理文档内容。

## 排序逻辑

## 排序配置

本插件支持通过设置自定义排序顺序。
配置项：`Sort Order`
类型：字符串
默认值：`empty, other, task_checked, task_unchecked`

可用类型：
*   `empty`: 空块（文本内容为空）
*   `other`: 其他块（普通文本、标题等）
*   `task_unchecked`: 未完成任务
*   `task_checked`: 已完成任务

## 默认规则

默认配置下 (`empty, other, task_checked, task_unchecked`)：

1.  **空块优先**：空行或无内容文本块排在最前。
2.  **普通内容**：普通文本块排在空块之后。
3.  **任务块后置**：所有任务块排在最后。
    *   **已完成任务**：在任务块中优先显示。
    *   **未完成任务**：排在所有内容的最底部。

所有同类项内部，均按照文本内容的**字典序**（A-Z）进行升序排列。
