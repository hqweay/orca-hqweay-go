# 支持块嵌入子项 (Embed Children)

这是一个专门优化“镜像块（Mirror Block）”展示体验的插件。启用后，允许带有特定标签的块，在作为镜像块嵌入到其他页面时，能自动展示其子块内容并应用无缝的样式。

## 功能特性

- **动态展示子项**：通常镜像块只展示引用的根块自身内容。开启该插件后，指定的块在作为镜像块时会自动展示它的全部子项。
- **自定义触发标签**：可以在设置中自由配置一个标签名（默认 `块嵌入子项`），只有带有该标签的块才会触发特殊样式。
- **样式无缝融合**：去除了默认的左侧边距及缩进线，让引用的树状块能够更加自然地融入当前文档的流式排版中。
