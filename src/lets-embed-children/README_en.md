# Embed Children

This is a plugin designed to optimize the display experience of "Mirror Blocks". When enabled, blocks with a specific tag will automatically display their children blocks and apply seamless styling when embedded as a mirror block in other pages.

## Features

- **Dynamic Children Display**: Normally, a mirror block only displays the referenced root block itself. With this plugin enabled, the specified blocks will automatically display all their children when used as a mirror block.
- **Custom Trigger Tag**: You can freely configure a tag name (default `块嵌入子项`) in the settings. Only blocks with this tag will trigger the special styling.
- **Seamless Styling**: Removes the default left margin and indentation lines, allowing the referenced tree block to blend more naturally into the current document's flow.
