# 📌 置顶块面板 (Pinned Blocks)

一款让你将重要块「置顶」到编辑器侧边栏，随时一键侧边打开的工具。

## ✨ 功能特性

- **标签驱动**: 为任意块打上配置的标签（默认 `#置顶`），该块即出现在置顶面板中。
- **displayName 支持**: 在标签属性中设置 `displayName`，自定义块在面板中的显示名称（与随机漫步插件一致）。
- **侧边栏按钮**: 在编辑器右侧边栏注册一个 📌 图钉按钮。
- **Hover 预览**: 鼠标悬停在图钉按钮上，自动弹出所有置顶块列表。
- **上次记忆**: 左键点击图钉按钮，直接在侧边面板重新打开上次点击的块；右键点击打开完整选择列表。
- **高亮标记**: 列表中高亮显示上次打开的块（图钉图标 + 「上次」标签）。
- **自定义标签名**: 可在设置中修改置顶标签名称。

## 🚀 使用方法

1. 在 Sub-plugin Settings 中启用 **Pinned Blocks** 插件。
2. 为你想置顶的块打上 `#置顶` 标签（可在设置中自定义）。
3. （可选）点击该标签，在属性面板中设置 `displayName` 为你想要的显示名称（如"项目导航"）。
4. 在编辑器右侧边栏找到 📌 图标：
   - **悬停**：查看所有置顶块列表，点击即可在右侧新开面板。
   - **左键单击**：直接重新打开上次访问的块。
   - **右键单击**：打开完整列表进行选择。
