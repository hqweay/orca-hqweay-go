# Remove Style

Quickly clean up note content formatting.

## Features

- **Remove Inline Styles**: Clear rich text styles like bold, highlight, etc.
- **Remove Links**: Convert links to plain text.
- **Remove Spaces**: Delete empty lines (without children).
- **Convert to Auto Headings**: Uniformly convert all levels of headings within the note to automatic hierarchy headings.

## Settings Configuration

You can customize which clean-up features are enabled in the sub-plugin settings panel:
- **Remove Inline Style** (enabled by default)
- **Remove Link Style** (enabled by default)
- **Remove Empty Lines** (enabled by default)
- **Convert to Auto Headings** (enabled by default)

Disabled features will not be displayed in the top headbar menu or the actions menu.
