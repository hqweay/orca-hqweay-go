# 🪐 Arc 侧边栏 (Arc Tabs)

为虎鲸笔记 (Orca) 深度定制的现代化标签页与侧边栏管理系统，提供媲美 Arc 浏览器的丝滑体验与高颜值界面。彻底告别杂乱的标签页，让多笔记、多页面的沉浸式协作更加高效。

## ✨ 核心特性

- **🪐 沉浸式多空间 (Spaces)**: 
  - 支持创建多个独立的空间（默认采用 🪐 等 Emoji），并在底部快捷栏快速切换。
  - 支持纯 Emoji、汉字或英文作为空间名称，底部空间图标完美兼容 Emoji 原生解析与渲染。
  - 各个空间之间的“已固定”标签完全独立，帮助你从底层隔离工作、学习与生活。

- **📌 强大的固定区域 (Pinned Tabs)**:
  - **标签化持久化**: 采用 `#ArcTab` 标签（可自定义）作为底层数据源，与双链生态完美融合。
  - **拖拽直达 (Drag & Drop)**: 选中编辑器中任意 Block 直接拖入侧边栏，即可实现秒级固定。
  - **网格/列表双布局**:
    - *Grid 网格模式*: 提供极致的空间利用率，采用类似移动端 App 的图标化展示。
    - *List 列表模式*: 经典的纵向列表，显示完整标题，鼠标悬浮自动浮现操作按钮。
  
- **🕒 丝滑的今日记录 (Today Tabs)**:
  - **实时同步感知**: 自动收集并记录当前激活以及最近浏览过的 Block，去重并自动过滤掉已固定的内容。
  - **安全防抖的关闭逻辑**: 在 Today 列表点击“关闭”，仅从视觉和历史记录中抹去，绝不强杀底层核心面板，完美保护你的工作现场；同时修复了底层状态轮询导致的“已关标签复活”现象。
  - **离线秒开缓存**: 标题和图标被巧妙地存入 `localStorage`，重启软件也能瞬间恢复，杜绝空白等待。

- **✨ 媲美原生的极致微动画**:
  - 为不同视图量身定制了纯 CSS 动画（列表模式的 `SlideIn` 平滑推挤展开，网格模式的 `ZoomIn` 轻巧弹出）。
  - 各个交互状态间（如 Hover、Active）的背景色、文字颜色切换均配置了 `0.3s ease` 的过渡曲线，彻底告别生硬突兀的闪烁与跳跃感。

## 🚀 快速上手

1. 在子插件设置中心开启 **Arc Tabs** 插件。
2. 调用 `Arc 侧边栏` 命令（或点击顶部图标）展开专属面板。
3. **固定页面**：
   - 将鼠标悬停在 Today 列表的任意项目上，点击出现的 📌 按钮。
   - 或直接按住编辑器中 Block 左侧的把手，将其拖入顶部的固定区。
4. **管理空间**：
   - 点击底部栏右侧的 `+` 号新建空间（强烈推荐使用单个 Emoji 命名！）。
   - 点击底部的不同图标即可瞬间切换你的专注流。

## ⚙️ 插件配置项

- **固定栏布局 (Layout)**: 自由选择 `grid`（网格）或 `list`（列表）以适应你的偏好。
- **关联标签名 (Tag Name)**: 默认绑定 `#ArcTab` 标签进行持久化，可以在设置中随时修改绑定。

## 🛠️ 技术实现亮点

- **全局状态代理**: 全面采用 `Valtio` 进行跨组件状态代理，实现 `0ms` 延迟的极速响应式 UI 更新。
- **解耦的导航系统**: 深度解耦了侧边栏展现逻辑与 Orca 底层状态树，精准拦截无效的更新。
- **纯粹的 CSS 魔法**: 拒绝引入臃肿的动画库，仅依靠精心调优的 CSS Keyframes 实现了接近原生级别的顺滑手感。
