# 🗂️ Arc 侧边栏 (Arc Tabs)

为虎鲸笔记提供媲美 Arc 浏览器的现代化标签页与侧边栏管理系统，让多笔记、多页面协作更加高效和流畅。

## ✨ 功能特性

- **多空间管理 (Spaces)**: 支持创建和切换不同的“空间”（如“个人”、“工作”等），各空间中的已固定标签页完全独立。
- **已固定标签页 (Pinned Tabs)**:
  - **标签化持久化**: 使用 `#ArcTab` 标签（名称可自定义）进行固定状态持久化，完美融入图谱与双链中。
  - **拖拽固定**: 支持直接把编辑器中的任何 Block 拖拽到侧边栏固定区域，松手即自动打标并固定。
  - **网格/列表双布局**:
    - **网格模式 (Grid)**：图标化紧凑网格展示。如果 Block 没有自定义 Emoji 图标，系统会自动提取标题的首个字符大写展示，视觉非常清爽。
    - **列表模式 (List)**：经典的标题加图标垂直列表。
  - **拖拽排序**: 在侧边栏中通过简单的拖拽即可对固定块进行自由排序，顺序会自动持久化在插件设置中。
  - **静默后台打标 Fallback**: 即使对未打开/非聚焦的 Block 进行 Pin/Unpin，插件也会在后台启动静默隐藏面板来确保打标命令 100% 成功写入数据库，用户无感知。
- **今天访问记录 (Today Tabs)**:
  - **当前活动面板**: 实时展示当前屏幕上所有处于打开状态的 Block 视图，确保活动工作区常驻。
  - **最近访问历史**: 自动记录并持久化最近访问过的 15 个 Block ID。
  - **合并去重**: Today 区域展示以上两者的并集去重结果（自动排除已 Pin 过的块）。
  - **标题/图标本地缓存**: 自动在 LocalStorage 中对最近 Block 的标题与自定义图标进行缓存，保证在重启软件或断网时能瞬间渲染出来，防止空白。
- **极致流畅体验**: 状态完全托管于 Valtio 全局状态代理，内存直操实现 UI 0ms 极速响应，并在后台自动与底层数据库最终同步。

## 🚀 使用方法

1. 在子插件设置中心开启 **Arc Tabs** 插件。
2. 点击顶部工具栏的文件夹图标 `folders`（或调用 `Arc 侧边栏` 命令）即可展开 Arc 侧边栏面板。
3. **固定 Block**：
    - 鼠标悬停在 Today 列表中的任意 Tab 上，点击右侧出现的 📌 按钮；
   - 或者是直接按住编辑器中任意 Block 左侧的拖动把手，将其拖入侧边栏的固定区释放。
4. **管理 Space**：
   - 在侧边栏顶部可以切换空间，右键点击空间名称可以新建、重命名或删除空间。
5. **设置布局**：
   - 在子插件设置中，可修改“固定栏布局”为**网格 (紧凑)**或**列表 (详细)**。
