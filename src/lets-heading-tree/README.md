## 功能介绍

Heading Tree 插件可以根据块的标题层级（H1-H4）自动重新组织块的嵌套结构。

### 主要功能

1. **选中块重排**：在编辑器中选中多个块，右键菜单中选择“标题重排”，插件将根据选中块中的标题层级自动调整它们的嵌套关系。
2. **整页重排**：通过顶部栏按钮或命令面板执行“重排当前页面”，自动整理当前页面根层级的所有块。

### 配置选项

在插件设置中，你可以配置 **顶部栏显示模式**：
- **操作菜单 (Actions Menu)**：仅在顶部栏的“更多操作”菜单中显示。
- **独立按钮 (Standalone Button)**：在顶部栏直接显示一个重排按钮。
- **两者 (Both)**：同时在菜单和顶部栏显示。

## 示例

重排前：
- H1 标题
- 文本内容
- H2 子标题
- 更多内容

重排后：
- H1 标题
  - 文本内容
  - H2 子标题
    - 更多内容
