# 🎲 随机漫步 (Random Walk)

一款专为虎鲸笔记打造的、基于标签的强大随机漫步工具。

## ✨ 功能特性

- **零配置侦测**: 为查询块或包含子节点的普通块打上“漫步标签”（如 `#随机漫步`），即被自动识别为独立的漫步频道。
- **双模式支持**:
  - **查询块**: 通过原生随机引擎自动发牌，无重复遍历。
  - **普通块**: 自动获取其子节点并进行纯内存乱序洗牌。
- **自定义命名**: 支持为漫步标签配置 `displayName` 属性，在频道列表中显示规整的频道名称。
- **右键即刻漫步**: 右键点击任意父块或查询块即可直接开启临时漫步，无需打标。
- **频道记忆**: 主按钮智能记忆上一次漫步的频道（包括临时漫步），一键继续未完的探索。

## 🚀 使用方法

1. 在虎鲸笔记设置 -> Sub-plugin Settings 中启用 **Random Walk** 插件。
2. 在你需要漫步的块（如查询出所有“书签”的查询块，或者包含几十句名人名言大纲的普通父块）上打上 `#随机漫步`（可以在插件设置中自定义该标签）。
3. 如果你想让下拉菜单中的名字更好看，可以点击标签，在属性面板中设置 `displayName` 为你想要的名字（例如“名人名言”）。
4. 点击顶部 Headbar 的 🎲 骰子图标即可漫游你上次看过的频道，或将鼠标悬浮在图标上以切换频道。
5. **临时漫步**: 在任意感兴趣的块上右键选择“随机漫步”，即可直接开始探索该块的内部世界。此时顶部的骰子图标也会自动记忆并锁定到该块。
