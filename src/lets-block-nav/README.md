# lets-block-nav

块结构导航面板插件 - Tana 风格的块导航和管理工具。

## 功能

- **子节点导航** - 显示当前编辑块的子节点列表
- **拖拽添加** - 从编辑器拖拽块到面板，作为子节点添加
- **拖拽排序** - 在面板中拖拽节点改变顺序
- **多选操作** - 支持 Ctrl/Shift 多选节点
- **批量删除** - 删除选中的节点
- **展开/折叠** - 控制节点显示

## 使用方法

1. 点击编辑器右侧工具栏的树形图标打开导航面板
2. 或使用命令 `Block Navigator: Open` 打开

## 操作说明

- **点击节点** - 在编辑器中导航到该块
- **Ctrl/Cmd + 点击** - 多选节点
- **Shift + 点击** - 范围选择
- **拖拽节点** - 改变节点顺序或作为子节点
- **从编辑器拖入** - 添加为当前块的子节点
