# Changelog

## 3.3.0

### Minor Changes

- c08ebf2: - 局部关系图谱：新增全局侧边栏局部关系图谱展示子插件
  - 支持实时跟踪编辑器导航轨迹，展示当前活动块及其足迹网络的局部拓扑结构
  - 提供对标签节点、父子大纲层级结构、常规显式引用的快捷视图菜单过滤
  - 支持右键点击边缘邻居节点，进行局部定向拓展或折叠收起
  - 采用物理衰减阻尼与静态静默拦截设计，动画过渡平滑防止剧烈晃动，并在无新邻居时进行静默拦截
  - 局部关系图谱：新增回放探索模式
    - 点击幻灯片图标进入回放模式，足迹节点按访问顺序以 900ms 间隔逐步呈现
    - 回放期间相机自动跟随最新节点，用户拖拽/缩放后暂停跟随
    - 退出回放时自动恢复录制状态，手动干预录制不受影响
  - 局部关系图谱：增强遍历路径可视化
    - 时序箭头长度 4 增至 10，并跟随主题色渲染，方向感知更清晰

## 3.2.0

### Features

- 编辑器层级折叠：新增精准控制主编辑器展示深度的子插件
  - 支持右键点击任意块，执行"展开至此同级"
  - 支持通过全局命令绑定快捷键，一键展开至第 1~5 层
- 大纲导航新增快照式层级展开功能，可一键展开至指定层级
  - 为高级用户注册了全局编辑器命令"大纲导航：展开至层级 1~5"以及"大纲导航：展开全部"
  - 优化交互反馈：大纲侧边栏底部展开按钮现支持多模式联动操作

## 3.1.1

### Improvements

- 新增大纲导航设置项：隐藏内置大纲。开启后在大纲导航面板激活时自动隐藏 Orca 内置的大纲面板，卸载时自动复原。

## 3.1.0

### Features

- 新增大纲树快捷过滤功能
  - 支持快捷点击「标题」「待办」「已办」，一键提取文档骨架与任务视图
  - 支持在搜索框输入指令查询，可与普通文本搜索组合使用
- 优化点击导航交互
  - 左键点击：编辑器平滑滚动定位到目标块
  - Alt/Option + 左键点击：编辑器下钻到该块
  - 右键点击：呼出上下文菜单，提供快捷转换等功能
